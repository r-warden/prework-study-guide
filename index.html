HTML

  • The head element contains information about the webpage.
  • The meta element is used to define various metadata.
  • The title element will appear in the browser tab and can be helpful in SEO.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Elements in a css file are referenced directly, classes start with a period.
  • Colors can be chosen semantically or with hexadecimal.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all mods in current branch to staging area
  • git commit -m "imperitive statement describing this commit": commits the staged code in the current branch with the quoted description
  • git pull origin branch-name: pulls latest version of specified branch from GitHub repo to local environment
  • git push origin branch-name: pushes changes in current branch to GitHub repo

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in shich a computer executes code in a script.
  • Chrome DevTools are helpful for making sure code runs correctly and debugging if it does not
